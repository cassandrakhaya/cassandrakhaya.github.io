Project Description

This is the current group project I am working on. It is a QR-based ordering system designed for restaurants. Customers will be able to scan the QR code assigned to their table, which will direct them to a mobile browser where they can view the menu and place their orders. The system will also include a worker-side interface, divided into server/cook and manager views. The manager will have the ability to edit the menu and view all placed orders. The cook will be able to view orders and notify servers when a specific order is ready to be served. The system is designed to help kitchen staff organize their tasks more efficiently. Additionally, a feature to track orders by status—such as new, in progress, and completed—is preferred.

The prototype link provided above showcases the first version I have created for the user side of this project. Currently, I am finalizing the worker-side interface. Once completed, I will transform these prototypes into a React and JavaScript-based frontend.
